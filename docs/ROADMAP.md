# Workflow & Roadmap

## Development Workflow

- **Run Dev Server**: `npm run dev`
- **Lint**: `npm run lint`
- **Build**: `npm run build`

## Future Roadmap

1. **Backend Integration**: Replace mock data with real API calls.
2. **Real-time Data**: Integrate WebSocket or polling for stock prices.
3. **Authentication**: Implement NextAuth.js or similar provider.
4. **Database**: Connect to PostgreSQL/Supabase.

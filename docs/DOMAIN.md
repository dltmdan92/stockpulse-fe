# Domain Guide - 투자 도메인 가이드

## 서비스 컨셉

StockPulse는 **투자 저널링**에 초점을 둔 포트폴리오 관리 도구입니다.
단순 자산 추적이 아닌, 모든 거래에 "왜?"를 기록하게 하여 투자 의사결정을 복기하고 개선하는 데 목적이 있습니다.

## 타겟 사용자

| 페르소나 | 특징 | 주요 사용 기능 |
|---------|------|---------------|
| 액티브 투자자 | 스윙/데이 트레이딩, 잦은 매매 | 거래 기록, 태그 분석 |
| 학습형 투자자 | 투자 실력 향상 목적 | 저널링, 성과 분석 |
| 시각형 투자자 | 차트/대시보드 선호 | 대시보드, 자산 추이 |

## 핵심 도메인 개념

### 포트폴리오 (Portfolio)
- **보유 종목 (Stock)**: 사용자가 현재 보유한 주식
- 종목 코드: 미국(AAPL, NVDA) / 한국(005930)
- 주요 지표: 수량, 평균매수가, 현재가, 수익률
- 섹터: Technology, Financials, Consumer Discretionary, Healthcare 등
- 국가: US, KR

### 거래 기록 (Trade) - 핵심 기능
모든 거래는 반드시 다음을 포함합니다:
- **매수/매도 (buy/sell)**: 거래 유형
- **메모 (memo)**: 투자 근거 (필수) - *이것이 StockPulse의 핵심 차별점*
- **태그 (tags)**: 전략 분류 태그 (예: #AI, #저점매수, #모멘텀, #실적시즌)
- **목표가 (targetPrice)**: 선택적
- **손절가 (stopLoss)**: 선택적

### 전략 태그 (Tags)
태그를 통해 전략별 성과를 분석할 수 있습니다:
- 어떤 전략이 높은 승률을 보이는지
- 어떤 전략이 평균 수익률이 높은지
- 전략별 거래 횟수와 추이

### 성과 분석 (Analysis)
| 지표 | 설명 |
|------|------|
| 승률 (Win Rate) | 수익 거래 비율 (%) |
| 평균 수익률 (Avg Return) | 거래 평균 수익률 (%) |
| 수익 팩터 (Profit Factor) | 총수익 / 총손실 |
| 평균 보유 기간 (Avg Holding Days) | 평균 보유일수 |
| 최대 수익 (Best Trade) | 단일 최고 수익 거래 (%) |
| 최대 손실 (Worst Trade) | 단일 최대 손실 거래 (%) |

### 시장 정보 (Market)
- 주요 지수: S&P 500, NASDAQ 100, KOSPI, Dow Jones
- 뉴스: 종목별 관련 뉴스 피드

## 통화 처리 규칙

| 구분 | 통화 | 포맷 | 예시 |
|------|------|------|------|
| 한국 주식 | KRW | 소수점 없음, 쉼표 | ₩75,400 |
| 미국 주식 | USD | 소수점 2자리, 쉼표 | $189.84 |
| 수익률 | % | 소수점 2자리, +/- 기호 | +19.20% |

## 데이터 유효성 규칙

| 필드 | 규칙 |
|------|------|
| 종목 코드 (symbol) | 비어있으면 안 됨 |
| 수량 (quantity) | 양의 정수 |
| 가격 (price) | 양수 |
| 거래 메모 (memo) | 필수, 비어있으면 안 됨 (핵심 기능) |
| 태그 (tags) | 문자열 배열, 각 항목 `#` 접두사 |
| 거래 유형 (type) | `'buy'` 또는 `'sell'` 만 허용 |
| 감정 (sentiment) | `'bullish'`, `'bearish'`, `'neutral'` 만 허용 |
